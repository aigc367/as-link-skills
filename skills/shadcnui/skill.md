# shadcn/ui

コピー&ペーストで使えるReact UIコンポーネント集。Radix UIとTailwind CSSベース

## インストール

```bash
npx shadcn@latest init
```

## 使用例

```bash
# コンポーネント追加
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add dialog

# 全コンポーネント追加
npx shadcn@latest add --all
```
