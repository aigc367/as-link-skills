# TypeScript

JavaScriptに型システムを追加したプログラミング言語。型安全性とIDEサポートを向上

## インストール

```bash
npm install -D typescript @types/node
```

## 使用例

```bash
# TypeScriptコンパイラ初期化
npx tsc --init

# TypeScriptファイルをコンパイル
npx tsc

# TypeScriptファイルを直接実行
npx tsx script.ts
```
