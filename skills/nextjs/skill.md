# Next.js

Reactベースのフルスタックフレームワーク。App Router、Server Components、APIルートなどの機能を提供

## インストール

```bash
npx create-next-app@latest
```

## 使用例

```bash
# 開発サーバー起動
npm run dev

# 本番ビルド
npm run build

# 本番サーバー起動
npm start
```
