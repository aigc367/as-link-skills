# Docker Helper

Dockerコンテナとイメージを管理するためのヘルパースキル

## インストール

```bash
npx skills add docker-helper
```

## 使用例

```bash
# コンテナ一覧を表示
docker-helper ps

# イメージをクリーンアップ
docker-helper cleanup

# コンテナログを表示
docker-helper logs <container-name>
```
